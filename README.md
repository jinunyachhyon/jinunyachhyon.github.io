### **Understand the Chirpy Directory Structure**
Familiarize yourself with the main folders and files in your repository:
- `_config.yml`: Global configuration settings for your website.
- `_posts`: Blog posts in Markdown format.
- `_layouts`: HTML templates for different pages.
- `_includes`: Reusable partial templates (e.g., header, footer).
- `assets`: Static files like images, CSS, and JavaScript.
- `_tabs`: Additional pages like About, Contact, etc.
